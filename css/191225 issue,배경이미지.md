### background

백그라운드 이미지가 나타나지않으면 F12를 눌러 확인한후

부모요소 를 확인한다 html - body - div

자식태그는 부모요소에 영향을 받는다

~~~
ex)

html, body{
    width:100%;
    height: 100%;
}


.background__01{
    width:100%;
    height:100%;
    background: url(/images/a.jpg) no-repeat;
    background-size: cover;
    background-position: center;
}
~~~
