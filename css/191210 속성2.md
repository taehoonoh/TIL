### animation

: 요소 애니메이션을 제어 작동한다 

~~~
ex)
     .box:hover{
        animation: hellomove 3s infinite        무한
        
        alternate;                              정방향재생 역재생
        }

        @keyframes hellomove{

            0%{

                width:100px;
                background: tomato;
            
            }

            75%{
                width:500px;
                background: dodgerblue;
            }

            100%{
                
               width:300px;
               background: yellowgreen;
            }
        }

+ 트랜지션은 0과 100%만 제어하지만 애니메이션은 더 디테일하게 제어가능 
~~~

 display:grid;
