## 목록 태그

+ ul과 ol안에는 li 태그만 가능     

+ dl 안에는 dt dd만 가능 

+ ul ol dl 서로 중첩,호환이 가능하다.

+ 단위:em(배)를 주로 쓴다(1.5~2)

뎁스(depth):연결 페이지? nav?



### 리스트 스타일 대표3
*list-style-position: inline outline 불릿 인라인 아웃라인

*list-style-image:url() 불릿이미지를 바꿔준다

*list-style:none 불릿을 없애준다


### dl태그 특징

+ dt(소제목) 내부 강조 span a 태그 정도만(헤딩태그랑 유사) 

+ dd(리스트) 내부에는 인라인요소
